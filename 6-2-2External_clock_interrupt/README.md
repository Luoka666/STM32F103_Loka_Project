# TIM 外部时钟脉冲计数实验

## 1. 故障现象
在使用对射式红外传感器作为外部时钟源（ETR）时，手动遮挡并松开一次传感器，计数器 `cnt` 并非单次递增，而是直接产生随机的大幅跳变。

## 2. 故障原因
* **硬件机械抖动**：传感器在触发边缘或受机械动作影响时，电平会产生高频毛刺和多次跳变。
* **触发过敏**：STM32 定时器的外部时钟输入极其灵敏，会将这些毛刺误判为多次有效的触发电平，导致计数异常。

## 3. 解决方案
通过配置定时器内部的 **数字滤波器** 来过滤毛刺信号。只有当输入电平保持平稳并达到指定的采样频率/点数时，硬件才认为是一次有效跳变。

## 4. 代码实现
在初始化函数中，通过修改 `TIM_ETRClockMode2Config` 的最后一个参数（Filter）来实现硬件滤波：

```c
// 核心配置：使用滤波器 (末尾参数 0x03)
// 该参数范围为 0x00-0x0F，数值越大滤波效果越强
TIM_ETRClockMode2Config(TIM2, TIM_ExtTRGPSC_OFF, TIM_ExtTRGPolarity_Inverted, 0x03);
